# Крестики-нолики на Windows Forms (C#)

Классическая игра с поддержкой двух режимов: стандартное поле 3×3 и расширенное 10×10.

## Возможности
- Два режима игры: 3×3 (победная линия 3) и 10×10 (победная линия 5)
- Визуализация победной линии
- Подсчет очков для каждого игрока
- Графический интерфейс с отрисовкой фигур (крестики и нолики)

## Технологии
- C#, Windows Forms
- Графическая отрисовка через GDI+ (метод OnPaint)
- Обработка событий мыши
- Двумерные массивы для хранения состояния поля


*Проект разработан в рамках изучения Windows Forms и работы с графикой.*
